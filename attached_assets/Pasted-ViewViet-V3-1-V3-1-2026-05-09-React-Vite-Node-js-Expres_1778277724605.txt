ViewViet — 跨境内容中台开发说明书 V3.1
版本：V3.1 | 交付日期：2026-05-09
技术栈：React (Vite) + Node.js Express + Supabase + Hostinger Business
品牌：ViewViet（原 VietHub）
域名：viewviet.net | falaw.pro | admin.viewviet.net | api.viewviet.net

一、品牌定位与命名规范
1.1 品牌架构
域名	品牌名	定位	目标用户
viewviet.net	ViewViet	跨境内容中台（学习+旅游+法律聚合入口）	出海企业、旅行者、语言学习者
falaw.pro	FaLaw Pro	专业法律博客（ASEAN Legal Hub）	企业法务、跨境律师、投资者
admin.viewviet.net	ViewViet Admin	后台管理系统	内容编辑、超级管理员
api.viewviet.net	ViewViet API	统一接口网关	内部服务、第三方合作
1.2 命名规范
•代码中所有 VietHub / viet_hub / VIETHUB 统一替换为 ViewViet / view_viet / VIEWVIET
•数据库表名、环境变量、CSS 变量保持小写 + 下划线
•用户可见文案统一使用 ViewViet（首字母大写，无空格）

二、系统架构总览
2.1 部署架构
用户端 (viewviet.net)          法律站 (falaw.pro)           管理端 (admin.viewviet.net)
    │                              │                              │
    └──────────────┬───────────────┘──────────────────────────────┘
                   │
           Hostinger Business
           ├── public_html/          → viewviet.net (静态 React)
           ├── public_html_falaw/    → falaw.pro (静态 React)
           ├── Node.js #1            → api.viewviet.net (Express API)
           └── Node.js #2            → admin.viewviet.net (React + Node SSR)
                   │
                   ▼
           Supabase (PostgreSQL + Storage + Auth)
2.2 技术选型理由
层级	技术	选型理由
前端框架	React 18 + Vite	构建速度快，Hostinger 静态部署友好
UI 组件	Tailwind CSS + Headless UI	原子化样式，暗色模式切换零成本
状态管理	Zustand + React Query	轻量，学习曲线低，适合中小项目
后端 API	Node.js Express	Hostinger Node.js 环境原生支持
数据库	Supabase PostgreSQL	免费额度充足，实时订阅，越南访问延迟可接受
文件存储	Supabase Storage	CDN 加速，替代 Hostinger 本地存储
认证	Supabase Auth (JWT)	内置用户系统，支持邮箱/手机号/第三方登录
部署	Hostinger + Cloudflare DNS	成本可控，支持 Node.js 进程

三、数据库设计（Supabase）
3.1 核心表结构
site_config — 站点配置（UI 实时配置中心）
CREATE TABLE site_config (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  config_group VARCHAR(50) NOT NULL, -- theme / layout / nav / footer / legal_theme
  config_key VARCHAR(100) NOT NULL,
  config_value TEXT,
  value_type VARCHAR(20) DEFAULT 'string', -- string / number / boolean / json / color
  is_active BOOLEAN DEFAULT true,
  updated_at TIMESTAMP DEFAULT NOW(),
  UNIQUE(config_group, config_key)
);
关键配置项： | config_group | config_key | 说明 | |————-|———–|——| | theme | primary_color | 主色 #0D7377 | | theme | accent_color | 强调色 #F2A900 | | theme | bg_color | 背景色 | | theme | text_color | 文字主色 | | layout | hero_modules | 首页模块排序 JSON | | nav | nav_items | 导航菜单 JSON 数组 | | footer | footer_content | 页脚富文本 | | legal_theme | legal_primary | falaw.pro 独立主题色 |
languages — 语言管理
CREATE TABLE languages (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  code VARCHAR(10) UNIQUE NOT NULL, -- vi / en / zh / ko
  name VARCHAR(50), -- Tiếng Việt
  name_en VARCHAR(50), -- Vietnamese
  flag_icon VARCHAR(10), -- 🇻🇳
  is_active BOOLEAN DEFAULT true,
  sort_order INT DEFAULT 0
);
words — 单词库
CREATE TABLE words (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  language_code VARCHAR(10) REFERENCES languages(code),
  word VARCHAR(100) NOT NULL, -- 目标语单词
  pronunciation VARCHAR(100), -- 音标/罗马字
  meaning_zh VARCHAR(255), -- 中文释义
  meaning_en VARCHAR(255), -- 英文释义
  meaning_vi VARCHAR(255), -- 越南语释义（新增）
  category VARCHAR(50), -- 机场/餐厅/医院/商务/购物...
  scene_tag VARCHAR(50), -- 细分场景标签
  audio_url TEXT, -- TTS 音频 URL
  image_url TEXT, -- 配图 URL
  example_sentence TEXT, -- 例句（目标语）
  example_translation TEXT, -- 例句翻译
  difficulty INT CHECK (difficulty BETWEEN 1 AND 5), -- 1=入门 5=精通
  is_published BOOLEAN DEFAULT false,
  created_by UUID REFERENCES auth.users(id), -- 新增：记录创建者
  created_at TIMESTAMP DEFAULT NOW()
);
scene_sentences — 场景句子
CREATE TABLE scene_sentences (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  language_code VARCHAR(10),
  scene_name VARCHAR(100), -- 机场值机 / 餐厅点餐 / 医院挂号...
  sentence TEXT NOT NULL, -- 目标语句子
  pronunciation VARCHAR(255), -- 发音标注
  translation_zh TEXT, -- 中文翻译
  translation_en TEXT, -- 英文翻译
  translation_vi TEXT, -- 越南语翻译（新增）
  audio_url TEXT,
  video_url TEXT, -- YouTube 切片链接
  related_word_ids UUID[], -- 关联单词
  difficulty INT CHECK (difficulty BETWEEN 1 AND 5),
  is_published BOOLEAN DEFAULT false,
  created_by UUID REFERENCES auth.users(id),
  created_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE complex_sentences (

CREATE TABLE legal_articles (

3.2 索引优化
-- 搜索优化
CREATE INDEX idx_words_category ON words(language_code, category);

CREATE INDEX idx_legal_articles_category ON legal_articles(category, is_published, published_at);

CREATE INDEX idx_activities_time ON activities(start_time, is_published);

-- 全文搜索（Supabase 内置）
CREATE INDEX idx_legal_articles_search ON legal_articles USING gin(to_tsvector('english', title || ' ' || content));
CREATE INDEX idx_words_search ON words USING gin(to_tsvector('english', word || ' ' || meaning_zh));

批量上传流程（单词/场景句/长难句）
用户操作：
1. 点击"批量导入" → 弹出模态框
2. 选择方式：
   a) 粘贴文字（每行一条）
   b) 上传 CSV/TXT 文件
   c) 上传 Excel（.xlsx）
3. AI 自动处理：
   - 识别中文内容
   - 自动翻译为目标语言
   - 自动分配分类（基于关键词匹配）
   - 自动分配难度（基于句子长度/词汇复杂度）
4. 预览界面：表格展示 AI 处理结果，可手动修正
5. 确认导入：一键保存至数据库
Word 文章导入流程
用户操作：
1. 法律/旅游管理页 → 点击"Word 导入"
2. 粘贴原文 或 上传 .docx/.txt
3. AI 自动排版：
   - 识别标题层级 → 生成 h1/h2/h3
   - 识别正文段落 → 生成 p
   - 识别列表 → 生成 ul/li
   - 识别图片 → 提示上传替换
   - 自动提取摘要
   - 自动识别分类（基于标题关键词）
4. 生成草稿 → 进入富文本编辑器微调
5. 发布或保存

十一、风险与规避
风险	影响	规避策略
Supabase 免费额度用完	数据库/存储中断	监控使用量，设置告警，准备迁移方案（PlanetScale / AWS RDS）
YouTube API 限流	视频搜索不可用	缓存搜索结果，降级为静态推荐列表
AI 翻译成本过高	运营亏损	批量翻译时限制并发，优先翻译高频内容，低频内容人工处理
越南访问速度慢	用户体验差	Cloudflare CDN + 图片 WebP 格式 + 懒加载
内容版权风险	法律纠纷	RSS 导入注明来源，原创为主，用户上传内容需审核
用户数据安全	泄露风险	Supabase Row Level Security (RLS)，敏感操作二次验证
